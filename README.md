# 🐱 猫咪博物馆

一个休闲合成 + 经营博物馆小游戏，纯前端单机，存档存在浏览器 LocalStorage。

## 玩法

- 三个馆：🐾 猫咪馆 / 🪨 自然馆 / 🏺 古董馆
- **合成**：相同物品两两合成升阶（L1 → L7），从猫粮到猫神祠、从骨片到始祖鸟、从残碗到传国玉玺
- **经营**：完成访客委托赚金币，购买装饰提升馆收入，升级展柜 / 解锁新馆
- **多线程**：合成 + 经营 + 装饰委托 三套循环互相喂养

## 本地运行

任意静态服务器即可，无任何后端依赖：

```bash
# 方式 1: Ruby 自带
ruby -run -e httpd . -p 8766

# 方式 2: Python
python3 -m http.server 8766

# 方式 3: Node
npx serve .
```

然后打开 http://localhost:8766

## 在线试玩

## 操作

- 拖拽 / 点击合成相同物品
- 顶部 HUD 切换三馆
- 右上角 **⚙️ 设置** → 重新开始（清空存档）
- 双击顶栏也可触发清档

## 技术栈

- 纯 HTML / CSS / 原生 JS，无构建步骤
- 单文件 game.js，约 3400 行
- 兼容移动端，iPhone 17 Pro Max 440px 一屏适配
